RTL Design Engineer

Design & Architecture · Remote · Full-time

Design and implement synthesisable RTL for complex ASIC/FPGA SoC blocks. You will own block-level architecture through to handoff to the physical design team, including synthesis constraints and timing closure.

Responsibilities

  • Write clean, well-constrained SystemVerilog/VHDL RTL
  • Define micro-architecture documents and RTL coding guidelines
  • Drive logic synthesis and static timing analysis closure
  • Collaborate with verification team on DV planning
  • Support DFT and PD hand-off packages

Requirements

  • 3+ years RTL design experience in ASIC or FPGA
  • Strong SystemVerilog coding and simulation skills
  • Familiarity with Synopsys DC or Cadence Genus
  • Experience with low-power design (UPF/CPF) is a plus
  • Strong communication and documentation habits

RTL Design Verification Engineer

Verification · Remote · Full-time

Develop and execute comprehensive verification plans for RTL blocks and sub-systems using UVM and industry-standard methodologies to achieve coverage closure.

Responsibilities

  • Build UVM testbenches (agents, sequences, scoreboards)
  • Write verification plans and coverage models
  • Develop functional and code coverage metrics
  • Debug RTL failures and file detailed bug reports
  • Drive regression automation and sign-off

Requirements

  • 3+ years UVM verification experience
  • Strong SystemVerilog and SVA skills
  • Familiarity with VCS/Xcelium or Questa
  • Experience with protocol VIPs (AXI, AHB, PCIe, etc.) a plus
  • Familiarity with formal property checking is desirable

AMS Verification Engineer

Verification · Remote · Full-time

Verify mixed-signal IP and SoCs using AMS simulation flows — Spice-to-RTL co-simulation, real-number modelling, and targeted stimulus generation for analog behavioural blocks.

Responsibilities

  • Develop real-number models (RNM) for analog blocks
  • Set up SPICE-RTL co-simulation environments
  • Define analog corner coverage and simulation plans
  • Interface with circuit designers for spec extraction
  • Drive mixed-signal regression and sign-off

Requirements

  • 3+ years AMS or mixed-signal verification experience
  • Strong Verilog-AMS or SystemVerilog RNM skills
  • Experience with Cadence Virtuoso or Spectre
  • Background in analog circuit fundamentals (ADC/DAC/PLL)
  • UVM knowledge is a plus

DFT Engineer

Design-for-Test · Remote · Full-time

Implement and optimise Design-for-Test architectures — scan insertion, BIST, ATPG pattern generation, and test coverage analysis — across complex ASIC designs from RTL to GDSII.

Responsibilities

  • Define DFT architecture and test strategy
  • Perform scan insertion and ATPG with Synopsys DFT tools
  • Implement MBIST and LBIST solutions
  • Analyse fault and test coverage metrics
  • Support ATE hand-off and production test development

Requirements

  • 3+ years ASIC DFT implementation experience
  • Hands-on with Synopsys DFT Compiler / Tessent
  • Experience with IEEE 1149.1 (JTAG) and 1500
  • Familiarity with low-power test challenges
  • Exposure to failure analysis flows is a plus

Embedded Systems Engineer

Embedded & Systems · Remote · Full-time

Develop firmware and low-level software for custom silicon and FPGA-based systems — bare-metal, RTOS, and Linux BSP development with a focus on hardware-software co-design.

Responsibilities

  • Write bare-metal and RTOS-based firmware in C/C++
  • Develop Linux BSPs and device drivers for custom SoCs
  • Implement peripheral and interface drivers (I2C, SPI, UART, PCIe)
  • HW-SW co-design and board bring-up support
  • Create system-level integration tests and debug utilities

Requirements

  • 3+ years embedded C/C++ development experience
  • Experience with ARM Cortex-M/A or RISC-V platforms
  • Linux kernel module or BSP development experience
  • Familiarity with hardware debugging tools (JTAG, oscilloscope)
  • Understanding of hardware interfaces and SoC register maps

Physical Design Engineer

Physical Design · Remote · Full-time

Implement block-level and chip-level physical design from floorplanning to GDSII — placement, CTS, routing, timing and power sign-off — using leading Cadence/Synopsys P&R tools.

Responsibilities

  • Floorplanning, power planning and pin assignment
  • Placement optimisation and CTS implementation
  • Signal routing and timing closure (STA with PrimeTime)
  • IR drop and electromigration analysis
  • LVS/DRC sign-off and GDSII tape-out preparation

Requirements

  • 4+ years physical design experience on tapeout projects
  • Proficient in Cadence Innovus or Synopsys ICC2
  • Strong STA skills with PrimeTime or Tempus
  • Experience with advanced nodes (16nm / 7nm / below) a plus
  • Familiarity with low-power multi-VT optimisation

Custom Layout Engineer

Custom IC / Layout · Remote · Full-time

Create high-performance custom analog and mixed-signal full-custom layout — standard cells, memory macros, and sensitive analog blocks — with a focus on matching, parasitics, and DRC/LVS cleanliness.

Responsibilities

  • Full-custom layout of analog/mixed-signal blocks
  • Device sizing for matching and parasitic optimisation
  • DRC and LVS verification and closure
  • Parasitic extraction and post-layout simulation
  • GDSII merge and tape-out preparation support

Requirements

  • 4+ years full-custom IC layout experience
  • Proficient in Cadence Virtuoso Layout Suite
  • Experience with Calibre DRC/LVS and Star-RC
  • Knowledge of fin/gate-all-around layout techniques is a plus
  • Foundry experience (TSMC / Samsung / GF) preferred
