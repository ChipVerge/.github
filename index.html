02
SystemVerilog RISC-V Microarchitecture

High-Speed 64-bit Multiplier Architecture

Challenge — The multiplier was the timing-sensitive arithmetic block in a 64-bit pipeline, and simpler iterative or radix-4 style options would have left too much reduction depth and routing pressure.

Outcome — Built a three-stage datapath with odd-multiple precomputation, nibble-wide crossbar selection, and hierarchical accumulation, while cleanly supporting MUL, MULH, MULHSU, MULHU, and MULW.

01
SystemVerilog Scheduling Hazard Control

Hazard-Aware Instruction Launcher

Challenge — A decode stage needed to keep execution supplied without launching instructions into register conflicts, blocking fences, or memory-order hazards.

Outcome — Built a queued launch block with per-slot hazard screening, propagated lock and memory state, and fixed-priority arbitration to emit one eligible instruction at a time.
